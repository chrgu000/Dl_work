使用说明

1. 系统：WindowsXP, Windows 7, Windows 8, Windows 10。

2. 浏览器：新版订单系统未做IE系列的兼容性测试，为了避免出现异常，强烈建议使用FireFox火狐浏览器( 点我下载 )或者Chrome谷歌浏览器。

3. 分辨率：推荐使用1280*768以上的分辨率，为了获得最佳体验请使用1920*1080的分辨率。

4. 网络：必须。
